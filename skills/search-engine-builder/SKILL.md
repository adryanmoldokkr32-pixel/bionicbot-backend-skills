# Search Engine Builder

This skill focuses on making information within an app easy to find with lightning-fast, relevant search results.

## Instructions

1. Choose the right search engine (SaaS like Algolia vs Self-hosted Meilisearch).
2. Index data from the database into the search engine.
3. Configure relevance rules, synonyms, and typo-tolerance.
4. Implement 'Search-as-you-type' UI on the frontend.
5. Handle real-time index updates when data changes.

## Examples

- "Integrate Algolia for a fast product search in my store."
- "Set up Meilisearch to provide full-text search for a documentation site."

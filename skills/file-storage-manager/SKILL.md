# File Storage Manager

This skill handles the logic for storing and retrieving files using services like AWS S3, Google Cloud Storage, or local storage.

## Instructions

1. Set up storage buckets/containers.
2. Implement secure upload policies (signed URLs, size limits).
3. Handle file renaming and path organization to prevent collisions.
4. Set up CDN integration for fast global file delivery.
5. Manage file permissions and public/private access levels.

## Examples

- "Set up an S3 bucket for user profile pictures."
- "Generate a signed URL for a private file download."

# Payment Gateway Integrator

This skill handles the complex logic of processing financial transactions and managing subscriptions.

## Instructions

1. Select the right payment provider for the region and use case.
2. Implement secure frontend checkouts (Stripe Elements, hosted pages).
3. Handle backend logic for payments, refunds, and disputes.
4. Sync subscription states with the application database.
5. Ensure PCI compliance and handle sensitive data securely.

## Examples

- "Integrate Stripe Checkout for a one-time product purchase."
- "Implement recurring billing for a SaaS with a free trial."

# Email SMTP Automator

This skill enables applications to send automated emails for sign-ups, password resets, and notifications.

## Instructions

1. Configure SMTP settings or API clients (SendGrid, Postmark, Resend).
2. Design HTML email templates with variables for personalization.
3. Implement retry logic for failed deliveries.
4. Handle unsubscribe requests and list management.
5. Track open rates and delivery statistics.

## Examples

- "Set up a 'Welcome' email that sends automatically when a user joins."
- "Build a password reset flow using Resend and Next.js."

# PDF/Invoice Generator

This skill focuses on creating dynamic, high-quality PDF files from data.

## Instructions

1. Use libraries like Puppeteer, React-PDF, or PDFKit.
2. Design a professional layout using HTML/CSS or specialized DSLs.
3. Inject dynamic data (customer info, line items, totals).
4. Handle multi-page tables and headers/footers.
5. Optimize PDF file size for storage and email delivery.

## Examples

- "Generate a monthly invoice PDF from a JSON object."
- "Create a custom report PDF with charts and data tables."

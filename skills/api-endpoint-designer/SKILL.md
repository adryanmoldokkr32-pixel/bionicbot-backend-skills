# API Endpoint Designer

This skill focuses on designing the communication layer between the frontend and backend using industry-standard protocols.

## Instructions

1. Define resource-based URLs (e.g., /api/v1/users).
2. Use correct HTTP methods (GET, POST, PUT, DELETE).
3. Design request/response schemas with clear error codes.
4. Implement filtering, pagination, and sorting for GET endpoints.
5. Draft API documentation using OpenAPI/Swagger format.

## Examples

- "Design a REST API for a task management system."
- "Create a GraphQL schema for a blog with nested comments."
